1. Open pgadmin
2. To create new database: Database -> Create -> Database -> Database[name] -> save
3. To initialize: right-click "Query Tool" of newly created database then copy and paste in code from students.sql and run the query
4. Run Interaction.py
5. Follow the instruction in the terminal (code written in main()) 

** For addStudents(), the enrollment date needs to be in this format: #-#-#, else it'll throw an error
** The first or the second run might throw an error. If it does, rerun and it'll work.
